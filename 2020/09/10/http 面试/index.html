http 面试

常用的 http 状态码

  • 1xx 服务器收到请求
  • 2xx 成功
    200请求成功
  • 3xx 重定向
    301 永久重定向
    302 临时重定向
    304 资源未被修改
  • 4xx 客服端服务
    404 文件没找到
    403 没有权限
  • 5xx 服务端错误
    500
    504 网关超时

http methods 方法

get 获取数据
post 新建数据；
patch/put 更新数据
delete 删除数据

http headers

  • 常见 headers
    request 请求 客户端往服务端发送的
    response 请求 服务端向客服端发送的
  • 缓存的 headers

http 缓存

强制缓存cache control

协商缓存

服务端缓存的策略
服务器判断客服端资源是否跟服务器资源一样
一致没有修改过返回304，否则返回200

  • last-modifid 资源最后修改的时间
  • etag 资源唯一标识（类似一个手纹）

    两者会优先使用 Etag
    last-modifid 只能精确到秒
